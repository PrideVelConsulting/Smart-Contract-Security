##  Import statements should always be placed at the top of the file.


---

### **Category Type** : QUALITY ASSURANCE / NON-CRITICAL


### **Explanation** : 

For Example


```javascript

//Yes:

// SPDX-License-Identifier: GPL-3.0
pragma solidity >=0.4.0 <0.9.0;

import "./Owned.sol";

contract A {
	// ...
}


contract B is Owned {
	// ...
}



//No:

// SPDX-License-Identifier: GPL-3.0
pragma solidity >=0.4.0 <0.9.0;

contract A {
	// ...
}


import "./Owned.sol";


contract B is Owned {
	// ...
}


```
