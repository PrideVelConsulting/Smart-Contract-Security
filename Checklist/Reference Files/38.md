## 38) Surround operators with a single space on either side.


---

### **Category Type** : QUALITY ASSURANCE / NON-CRITICAL


### **Explanation** : 


```javascript

 //Yes:
   	    
   		x = 3;
       	x = 100 / 10;
       	x += 3 + 4;
       	x |= y && z;
   	    
//No:   

   		x=3;
       	x = 100/10;
       	x += 3+4;
       	x |= y&&z;


```
