##  Operators with a higher priority than others can exclude surrounding whitespace in order to denote precedence.




---

### **Category Type** : QUALITY ASSURANCE / NON-CRITICAL


### **Explanation** : 


```javascript

//Yes:
   	    
   		x = 2**3 + 5;
       	x = 2*y + 3*z;
       	x = (a+b) * (a-b);
 
//No:   

   	    x = 2** 3 + 5;
      	x = y+z;
      	x +=1;
   		

```
